# POE2GPS Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Turn a fork of Sikaka/POE2Radar into POE2GPS — a strictly read-only PoE2 navigation overlay — by removing the auto-flask keystroke subsystem and poe.ninja pricing, grafting NattKh's process-randomization, and enforcing a compliance gate that fails the build on any input-emission or process-write API.

**Architecture:** Three .NET projects (`Core` = read-only memory + offsets, `Overlay` = render loop + dashboard + entry point, `Research` = dev-only, never shipped). Changes are confined to `Overlay` plus one new pure helper in `Core` and a test project. A PowerShell compliance gate (built first, RED→GREEN) drives the removals; the radar/atlas/route-planning read paths are untouched.

**Tech Stack:** C# / .NET 10 (`net10.0-windows`, x64), `.slnx` solution, Vortice.Direct2D1, Direct2D layered-window overlay, `System.Net.HttpListener` localhost dashboard, xUnit (new), PowerShell 7 build/gate scripts, GitHub Actions.

## Global Constraints

Every task implicitly includes these (copied verbatim from the spec + verified toolchain):

- **Read-only only.** No `SendInput`, `keybd_event`, `mouse_event`, `PostMessage`/`SendMessage`-to-game, `SetCursorPos`-for-input, `SendKeys`, input-simulator libs. No `WriteProcessMemory`, `NtWriteVirtualMemory`, `VirtualAllocEx`, `VirtualProtectEx`, `CreateRemoteThread`, `QueueUserAPC`, `SetWindowsHookEx`-injection. `OpenProcess` access mask stays `PROCESS_VM_READ | PROCESS_QUERY_(LIMITED_)INFORMATION` — never `PROCESS_VM_WRITE`/`PROCESS_VM_OPERATION`. No poe.ninja/poe2scout/MCP network calls.
- **Target:** `net10.0-windows`, `Platforms`/`PlatformTarget` `x64`, all builds/tests on Windows.
- **Solution is `.slnx`** (XML) — use `dotnet build POE2Radar.slnx` and `dotnet sln POE2Radar.slnx add …`.
- **`TreatWarningsAsErrors=true`** on every src project — no orphaned fields, unused usings, or dead private helpers may remain after any edit.
- **`Nullable=enable`, `ImplicitUsings=enable`, `LangVersion=latest`, `AllowUnsafeBlocks=true`** (Overlay/Core).
- **Only external NuGet dep is `Vortice.Direct2D1` `3.*`** (Overlay). Do not add runtime dependencies; the randomization graft must not pull in WinForms.
- **License stays MIT;** preserve `LICENSE`; credit Sikaka, NattKh, GameHelper2, contributors.
- **Build command:** `dotnet build POE2Radar.slnx` (from repo root). **Gate command:** `pwsh scripts/compliance-gate.ps1`. **Test command:** `dotnet test POE2Radar.slnx`.
- Reference clones for porting live at `.reference/sikaka` and `.reference/nattkh` (gitignored, never committed).

```powershell
#requires -version 7
[CmdletBinding()]
param(
  [string]$Root = (Resolve-Path "$PSScriptRoot/..").Path,
  [switch]$SelfTest
)
$ErrorActionPreference = 'Stop'

# Forbidden Win32/.NET symbols. Reads (ReadProcessMemory/NtReadVirtualMemory/VirtualQueryEx/
# OpenProcess/module-enum) are ALLOWED and deliberately absent from these lists.
$writeSymbols = @(
  'WriteProcessMemory','NtWriteVirtualMemory','ZwWriteVirtualMemory','VirtualAllocEx',
  'VirtualProtectEx','VirtualFreeEx','NtProtectVirtualMemory','CreateRemoteThread','CreateRemoteThreadEx',
  'RtlCreateUserThread','QueueUserAPC','NtQueueApcThread','SetWindowsHookEx','SetWindowsHookExW',
  'NtMapViewOfSection','ZwMapViewOfSection','PROCESS_VM_WRITE','PROCESS_VM_OPERATION','PROCESS_CREATE_THREAD'
)
$inputSymbols = @(
  'SendInput','keybd_event','mouse_event','PostMessage','PostMessageW','PostMessageA',
  'SendMessage','SendMessageW','SendMessageA','SendMessageTimeout','SendNotifyMessage',
  'SetCursorPos','BlockInput','SetKeyboardState'
)
# Word-boundary regex so managed File.WriteAllText / Console.WriteLine never match the exact Win32 names.
$pattern = '(?<![A-Za-z_])(' + (($writeSymbols + $inputSymbols) -join '|') + ')(?![A-Za-z_])'

# Allowlist of approved benign hits: "relpath:line:Symbol | justification"
$allow = @{}
$allowlistPath = Join-Path $PSScriptRoot 'compliance-allowlist.txt'
if (Test-Path $allowlistPath) {
  foreach ($l in Get-Content $allowlistPath) {
    if (-not $l -or $l.TrimStart().StartsWith('#')) { continue }
    $left = ($l -split '\|')[0].Trim()
    if ($left) { $allow[$left] = $true }
  }
}

function Scan([string]$dir) {
  $hits = @()
  $files = Get-ChildItem -Path $dir -Recurse -Filter *.cs -File |
    Where-Object { $_.FullName -notmatch '[\\/](bin|obj)[\\/]' -and $_.FullName -notmatch '[\\/]POE2Radar\.Research[\\/]' }
  foreach ($f in $files) {
    $n = 0
    foreach ($line in [System.IO.File]::ReadLines($f.FullName)) {
      $n++
      foreach ($m in [regex]::Matches($line, $pattern)) {
        $rel = ([System.IO.Path]::GetRelativePath($Root, $f.FullName)) -replace '\\','/'
        if (-not $allow.ContainsKey("$rel`:$n`:$($m.Value)")) {
          $hits += [pscustomobject]@{ File = $rel; Line = $n; Symbol = $m.Value; Text = $line.Trim() }
        }
      }
    }
  }
  return ,$hits
}

if ($SelfTest) {
  $tmp = Join-Path ([System.IO.Path]::GetTempPath()) ("gate-selftest-" + [guid]::NewGuid())
  New-Item -ItemType Directory -Path $tmp | Out-Null
  Set-Content "$tmp/Bad.cs"  'class X { void M(){ SendInput(1, null, 0); } }'
  Set-Content "$tmp/Good.cs" 'class Y { void M(){ ReadProcessMemory(h, a, b, c, out var n); } }'
  $bad = Scan $tmp
  Remove-Item $tmp -Recurse -Force
  if ($bad.Count -ne 1 -or $bad[0].Symbol -ne 'SendInput') {
    Write-Error "Self-test FAILED: expected exactly one SendInput hit, got $($bad.Count)."; exit 2
  }
  Write-Host "Compliance gate self-test PASSED (flags SendInput, ignores ReadProcessMemory)." -ForegroundColor Green
  exit 0
}

$violations = Scan "$Root/src"

# OpenProcess access-mask check: OpenProcess is allowed, but never with write/operation access.
$openBad = @()
$srcFiles = Get-ChildItem "$Root/src" -Recurse -Filter *.cs -File |
  Where-Object { $_.FullName -notmatch '[\\/](bin|obj)[\\/]' -and $_.FullName -notmatch '[\\/]POE2Radar\.Research[\\/]' }
foreach ($hit in (Select-String -Path $srcFiles.FullName -Pattern 'OpenProcess')) {
  if ($hit.Line -match 'PROCESS_VM_WRITE|PROCESS_VM_OPERATION') { $openBad += $hit }
}

if ($violations.Count -gt 0 -or $openBad.Count -gt 0) {
  Write-Host "COMPLIANCE GATE: FAIL" -ForegroundColor Red
  foreach ($v in $violations) { Write-Host ("  {0}:{1}  {2}   {3}" -f $v.File, $v.Line, $v.Symbol, $v.Text) }
  foreach ($o in $openBad)   { Write-Host ("  OpenProcess requests write access: {0}" -f $o.Line.Trim()) }
  exit 1
}
Write-Host "COMPLIANCE GATE: PASS — no input-emission or process-write symbols in shipped source." -ForegroundColor Green
exit 0
```

```csharp
namespace POE2Radar.Core.Stealth;

/// <summary>
/// Generates a pronounceable, identity-neutral random name (consonant/vowel alternation, 1-2 words).
/// Used to launch the overlay under a random hardlink name and to name the overlay window.
/// </summary>
public static class RandomName
{
    private const string Vowels = "eioau";
    private const string Consonants = "bcdfghjklmnpqrstvwxyz";

    public static string Generate(Random? rng = null)
    {
        rng ??= Random.Shared;
        var parts = new List<string>();
        for (var w = 0; w < rng.Next(1, 3); w++)
        {
            var len = rng.Next(5, 9);
            var ch = new char[len];
            for (var i = 0; i < len; i++)
                ch[i] = (i % 2 == 0) ? Consonants[rng.Next(Consonants.Length)] : Vowels[rng.Next(Vowels.Length)];
            ch[0] = char.ToUpper(ch[0]);
            parts.Add(new string(ch));
        }
        return string.Join("", parts);
    }
}
```

**Files:**

**Interfaces:**
- Produces: `pwsh scripts/scrub-strings.ps1 -ExePath <exe>` same-length-replaces identifying tokens in the published exe; `-SelfTest` proves it; `publish.ps1` runs it on an uncompressed single-file build so the scrub reaches embedded strings.

> **Scope note (honest):** the scrub targets clearly-safe identity literals (author/handle, repo URLs, "Sikaka"/"NattKh" credit strings, and the console banner — already neutralized in source). It does **not** scrub `POE2Radar` from .NET *type metadata* (namespaces/type names), because those are used by the runtime and same-length-blanking them risks reflection/exception/serialization breakage. Fully removing the namespace token would require a project-wide namespace rename — available as an optional follow-up, not done here. Same-length replacement is mandatory: changing byte lengths corrupts the single-file bundle's offset tables.

- [ ] **Step 1: Write `scripts/scrub-strings.ps1` with a self-test**

```powershell
#requires -version 7
[CmdletBinding()]
param(
  [string]$ExePath,
  [string[]]$Tokens = @('Sikaka','NattKh','github.com/Sikaka','github.com/NattKh'),
  [switch]$SelfTest
)
$ErrorActionPreference = 'Stop'

function Replace-Bytes([byte[]]$buf, [byte[]]$needle, [byte[]]$filler) {
  $n = $needle.Length; $hits = 0
  for ($i = 0; $i -le $buf.Length - $n; $i++) {
    $match = $true
    for ($j = 0; $j -lt $n; $j++) { if ($buf[$i + $j] -ne $needle[$j]) { $match = $false; break } }
    if ($match) { for ($j = 0; $j -lt $n; $j++) { $buf[$i + $j] = $filler[$j] }; $hits++; $i += $n - 1 }
  }
  return $hits
}

function Scrub([string]$path, [string[]]$tokens) {
  $bytes = [System.IO.File]::ReadAllBytes($path)
  $count = 0
  foreach ($t in $tokens) {
    $filler = 'x' * $t.Length
    foreach ($enc in @([Text.Encoding]::ASCII, [Text.Encoding]::Unicode)) {
      $count += Replace-Bytes $bytes $enc.GetBytes($t) $enc.GetBytes($filler)
    }
  }
  [System.IO.File]::WriteAllBytes($path, $bytes)
  return $count
}

if ($SelfTest) {
  $tmp = [System.IO.Path]::GetTempFileName()
  [System.IO.File]::WriteAllBytes($tmp, [Text.Encoding]::Unicode.GetBytes('hello Sikaka world'))
  [void](Scrub $tmp @('Sikaka'))
  $after = [Text.Encoding]::Unicode.GetString([System.IO.File]::ReadAllBytes($tmp))
  Remove-Item $tmp -Force
  if ($after -match 'Sikaka') { Write-Error 'scrub self-test FAILED'; exit 2 }
  Write-Host 'scrub self-test PASSED'; exit 0
}

if (-not (Test-Path $ExePath)) { Write-Error "exe not found: $ExePath"; exit 1 }
$n = Scrub $ExePath $Tokens
Write-Host "scrubbed $n occurrence(s) in $ExePath"
$raw = [System.IO.File]::ReadAllText($ExePath, [Text.Encoding]::ASCII)
foreach ($t in $Tokens) {
  if ($raw -match [regex]::Escape($t)) { Write-Error "token '$t' still present after scrub"; exit 1 }
}
exit 0
```
