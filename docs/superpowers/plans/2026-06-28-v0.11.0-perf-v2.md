# v0.11.0 Performance v2 Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Land the 8 deferred Bucket-2 memory-read optimizations + an RPM-rate readout as one behavior-preserving v0.11.0 release that measurably cuts ReadProcessMemory syscalls/sec and per-tick allocations on the world hot path.

**Architecture:** All work is read-side refactoring inside `POE2Radar.Core/Game/Poe2Live.cs` (the live reader) plus thin plumbing in `RadarApp.cs` (world-tick call swap + RPM rate) and `Web/ApiServer.cs` (`rpmPerSec` in `/state`). The RPM counter already exists (`MemoryReader.ReadCount`); we only add a rate helper and surface it. No new offsets, no behavior change to what is drawn.

**Tech Stack:** C# / .NET 10 (`net10.0-windows`, x64), xUnit (Core-only test project `tests/POE2Radar.Tests`), `System.Buffers.ArrayPool`, `System.Threading.Interlocked`.

## Global Constraints

- **Strictly read-only.** Only `OpenProcess` (read) + `ReadProcessMemory`/`NtReadVirtualMemory`. NO `WriteProcessMemory`/`VirtualProtectEx`/`CreateRemoteThread`, NO input emission (`SendInput`/`PostMessage`/`keybd_event`). The RPM counter only *counts*.
- **No new offsets.** `src/POE2Radar.Core/Game/Poe2Offsets.cs` is unchanged. All offsets used already exist.
- **No Core→Vortice dependency.** Core stays renderer-free.
- **Behavior-preserving.** Entities, rarity, hostility coloring, HP bars, map shift/zoom/visibility, item mods, terrain, and player vitals must be **identical** before/after. Any change to the drawn result is a regression.
- **Gates stay green:** `scripts/compliance-gate.ps1` and `scripts/scrub-strings.ps1 -SelfTest` must pass. `src/POE2Radar.Overlay/Program.cs` console title is untouched.
- **README badge stays `0.5.4`.** Version bumps to `0.11.0` in `src/POE2Radar.Overlay/POE2Radar.Overlay.csproj`.
- **Concurrency:** three independent `Poe2Live`/`MemoryReader` instances (`_live` world, `_liveRender` render, `_liveApi` API). Per-instance buffers/caches are NOT thread-safe; only touch the instance owned by the calling thread. The `_world`/`_state` snapshots are published via volatile reference swap and read lock-free.

```csharp
/// <summary>Resolve several component addresses for one entity in a single bucket read.
/// results[i] is the component address for names[i], or 0 if absent. names.Length == results.Length.</summary>
private void ResolveComponents(nint entity, ReadOnlySpan<string> names, Span<nint> results)
{
    for (var i = 0; i < results.Length; i++) results[i] = 0;

    var details = Ptr(entity + Poe2.Entity.EntityDetailsPtr);
    if (details == 0) return;
    var lookup = Ptr(details + Poe2.EntityDetails.ComponentLookUpPtr);
    if (lookup == 0) return;
    if (!_reader.TryReadStruct<StdVector>(entity + Poe2.Entity.ComponentList, out var compList)) return;
    var compCount = ((long)compList.Last - (long)compList.First) / 8;
    if (compCount is <= 0 or > 256) return;

    var bFirst = Ptr(lookup + Poe2.ComponentLookUp.NameAndIndexBucket);
    if (!_reader.TryReadStruct<nint>(lookup + Poe2.ComponentLookUp.NameAndIndexBucket + 8, out var bLast)) return;
    var entries = ((long)bLast - (long)bFirst) / Poe2.ComponentLookUp.EntryStride;
    if (bFirst == 0 || entries is <= 0 or > 256) return;

    var span = (int)(entries * Poe2.ComponentLookUp.EntryStride);
    if (_reader.TryReadBytes(bFirst, _compBucketBuf, span) < span) return;   // ONE bulk read of the whole bucket

    var remaining = names.Length;
    for (long i = 0; i < entries && remaining > 0; i++)
    {
        var off = (int)(i * Poe2.ComponentLookUp.EntryStride);
        var namePtr = (nint)BitConverter.ToInt64(_compBucketBuf, off);
        var index = BitConverter.ToInt32(_compBucketBuf, off + 8);
        if (index < 0 || index >= compCount) continue;
        var name = _reader.ReadStringUtf8(namePtr, 32);   // one read per entry, shared across all targets
        for (var t = 0; t < names.Length; t++)
        {
            if (results[t] != 0 || names[t] != name) continue;
            results[t] = Ptr(compList.First + (nint)(index * 8));
            remaining--;
        }
    }
}

private nint ResolveComponent(nint entity, string name)
{
```
