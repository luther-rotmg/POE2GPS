# v0.20.0 Manual Smoke Checks (pre-RC tag)

Run all seven against a live POE2 international client before cutting `v0.20.0-rc.1`.

## 1. `/map` — Firefox and Chrome, live zone

- Enable `EnableWebMap`, restart, open `http://localhost:7777/map`.
- Verify iso-tilt matches the game's minimap orientation.
- Verify player marker centred; facing triangle rotates smoothly while running; degenerates to a circle when standing still.
- Move through a zone: fog reveals a disc around the player.
- Toggle GPS mode (button or `G` key): fog vanishes, full walkable visible; toggle back — fog returns; already-revealed area stays transparent.
- Atlas nodes appear on the Endgame Atlas view (once bundled icons are populated).
- Monoliths show ring + holes count + best reward label; collected ones dim.
- Landmarks show small diamonds with labels.

## 2. Terrain orientation parity

Open a known-orientation zone (e.g. Ogham Village entrance). Compare the browser `/map` terrain outline to an in-game screenshot. If mirrored, revisit the `applyIsoTransform` signs in `map.js`.

## 3. `/obs` in OBS Browser Source

- Enable `EnableWebObs`, restart, add a Browser Source in OBS at 1920×1080 pointing to `http://localhost:7777/obs`.
- Verify transparent background, no HUD chrome, smooth motion at 60 fps recorded.
- Add a second source at `http://localhost:7777/obs?gps=1` — verify GPS mode active on that scene.

## 4. Zone-change stress

Transition through 5 zones back-to-back. Verify:
- `/api/map` doesn't stall `/state` (server multithread).
- Fog resets on each zone.
- Atlas + landmarks re-fetch cleanly.

## 5. Backpressure

- Open `/map`; DevTools → Performance → CPU throttling 6× slowdown.
- Verify server-side drops occur (check server console for drop counters) without disconnecting other healthy tabs.
- Verify a truly stuck tab (paused JS) is disconnected within ~1 s.

## 6. Kill switch

- Toggle `EnableWebMap = false`, restart.
- Verify `/map` returns 404, `/stream` returns 404, `/api/map` returns 404.
- Debug build's `zero-cost-when-off invariant violated` assertion does NOT fire.

## 7. Terrain edge parity (JS port vs C# reference)

- Open `/map` in Chrome, DevTools console.
- Fetch the `terrain-small.bin` fixture: `const bin = await (await fetch('/api/test-fixture/terrain-small.bin')).arrayBuffer();`
  (this endpoint is not in v0.20.0 — for the parity check, base64-encode `tests/POE2Radar.Tests/fixtures/terrain-small.bin` locally and paste as `atob('...')`).
- Call `window.__poe2gpsBuildEdges(walkable, 24, 24)` and byte-compare to `terrain-small-expected-edges.bin`.
- Any mismatch = the JS port has drifted from the C# reference; fix before RC.

## 8. Auto-updater sanity

- Cut a local `v0.20.0-rc.1-test` tag, run `AutoUpdater` against a mock release URL.
- Verify: fresh binary swap succeeds; one-generation rollback works.
