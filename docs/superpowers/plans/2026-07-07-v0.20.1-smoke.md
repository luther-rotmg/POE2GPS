# v0.20.1 Manual Smoke Checks (pre-RC tag)

Run all nine against a live POE2 international client before cutting `v0.20.1-rc.1`.

## 1. Vitals offsets read cleanly
- Launch overlay in-game; confirm HP/Mana/ES bars track through a full campaign zone with no drift notices in the console.
- Confirm the README support badge matches the current PoE2 patch.

## 2. Path polylines on `/map`
- Enable `EnableWebMap`, restart, open `http://localhost:7777/map`.
- Enter an atlas map with a computed route (any Waystoned map with tagged targets); confirm pathBlue polylines render between fog and landmarks layers.
- Verify path smoothly updates as the player moves along it.

## 3. Delta entities behaviour
- Open `/map` in Chrome DevTools → Network → filter `/stream`.
- First message should show `full: true` and a populated `entities` array.
- Second message should show `full: false` and `entitiesDelta: {add, upd, del}` blocks.
- HUD entity count should stay in sync (add/upd/del math consistent with what the eye sees).

## 4. Zone-change reset
- Cross a zone transition; confirm the very next SSE message after `area` changes is a full snapshot again.
- Confirm `map.js` clears its `state.entities` Map on the transition.

## 5. Preview channel dry run
- ⚙️ Settings → set UpdateChannel = "preview", restart.
- Confirm the update check hits `/releases` and picks the newest prerelease. If no prerelease exists, confirm it falls back to `/releases/latest` behavior gracefully.

## 6. Custom UpdateUrl dry run
- ⚙️ Settings → paste `https://api.github.com/repos/luther-rotmg/POE2GPS/releases` as UpdateUrl.
- Confirm the resolved URL uses the override (verify via console log or `--diag`).
- Clear the field, restart, confirm default URL restored.

## 7. Legacy settings file migration
- Copy your `config/radar_settings.json` from a v0.20.0 install into the v0.20.1 install directory (or use `tests/POE2Radar.Tests/fixtures/settings-v0.20.0-seeded.json`).
- Launch overlay; confirm no seeds re-fire (e.g., landmarks pack doesn't re-seed if it was seeded in v0.20.0).
- After first save, verify the legacy bool fields are omitted from the on-disk JSON and `AppliedMigrations` contains the corresponding string keys.

## 8. Dashboard hygiene
- Open `http://localhost:7777/`; confirm Session panel renders styled (has a bordered card with title, not raw text).
- Type in ⚙️ Settings search; confirm collapsed cards don't leak matches into the results.
- Visually spot-check danger/muted/bg colors are unchanged from v0.20.0.

## 9. Blank-canvas regression check
- Reload `/map` several times in quick succession.
- Drag the browser window between monitors (if available) to exercise window resize events.
- Verify canvas renders normally and doesn't show a black screen except during actual brief resize transitions.
